# 사용자 시나리오

## 개요

| 항목 | 내용 |
|------|------|
| 서비스 | 노화우 (老하우) |
| 대상 화면 | HOME / RECORD / SHARE_SHEET / ARCHIVE / DETAIL |
| 작성일 | 2026-05-17 |

> 화면 5종 기반 핵심 플로우 4개 — 기록·저장(A) · 공유(B) · 또래 게시판 열람(C) · 캘린더→상세 열람(D).

---

## 시나리오 1. 부모의 노하우 기록 (Flow A)

> 부모 사용자가 사진·음성·제목으로 노하우를 기록하고, [저장만] 또는 [공유] 분기로 마무리한다.

### 전제 조건
- 사용자(5·60대 부모)가 앱이 설치되어 있고 마이크 / 카메라 권한을 허용한 상태
- 인터넷 연결이 가능한 환경

### 플로우

**기록 화면 진입**
1. 사용자: 하단 탭 "기록" 으로 RECORD 화면 진입

**제목 / 사진**
2. 사용자: 상단 텍스트 필드에 제목 입력 (예: "상추 모종 심기")
3. 사용자: 사진 영역 탭 → 촬영 또는 갤러리 선택

**음성 기록**
4. 사용자: 빨간 원 버튼 길게 눌러 녹음 시작 → 발화 → 손 떼면 종료
5. 시스템: 녹음 UX 수행 후 STT 결과를 받은 척 클라가 `"안녕하세요"` 로 하드코딩 보관. 서버에는 본문 전송하지 않고 길이(초) 만 보냄. HOME/DETAIL 본문 영역에 이 하드코딩 텍스트 노출

**액션 분기**
6. 사용자: [저장만] 또는 [공유] 중 하나 탭
    - [저장만]: 시스템이 노하우 카드를 비공개로 저장 (isShared=false)
    - [공유]: 공유 시트(SHARE_SHEET) 오픈 → 시나리오 2 로 이어짐

**실패 처리**
- 등록 실패 시 토스트로 재시도 안내, 입력값 유지

## 시나리오 2. 공유 채널 선택 (Flow B)

> RECORD 또는 DETAIL 에서 공유를 누르면 시트가 뜨고, 사용자는 3개 채널 중 하나를 선택한다.

### 전제 조건
- 시나리오 1 의 [공유] 또는 DETAIL 의 [공유하기] 진입

### 플로우

1. 시스템: 직전 화면을 dim 처리하고 공유 시트(SHARE_SHEET) 슬라이드업
2. 사용자: 채널 선택
    - **카카오톡으로 보내기** (추천 배지): 카카오 공유 SDK 호출, 자녀·가족방·친구에게 전송
    - **링크 복사**: 카드 공유 URL 을 클립보드에 복사 → 토스트 안내
    - **또래 게시판에 공유**: 노하우 카드를 isShared=true 로 게시 (RECORD 진입 시점이면 등록과 동시에)
3. 사용자: 시트 외부 탭 → 시트 닫힘 (취소)

> 서버 API 가 관여하는 분기는 "또래 게시판에 공유" 1건. 나머지 2개는 클라 단독 처리.

---

## 시나리오 3. 또래 게시판 열람 (Flow C)

> 또래 부모 사용자들이 서로 공유한 노하우 카드를 갤러리 형태로 둘러본다.

### 전제 조건
- HOME 화면 진입
- 또래 게시판에 isShared=true 로 등록된 카드가 1건 이상 존재

### 플로우

1. 사용자: 하단 탭 "홈" 으로 HOME 진입
2. 시스템: 또래 노하우 카드 목록을 최신순 2×2 그리드로 노출
    - 카드: 사진(또는 이모지) · 작성자 이름·나이 · 본문 스니펫 · 좋아요수 · 댓글수 · 음성 길이
3. 사용자: 스크롤 → 무한 스크롤로 다음 페이지 자동 로딩
4. 사용자: (MVP 미구현) 카드 탭

> MVP 범위에서 HOME 카드 탭 → 상세 페이지 이동은 **구현하지 않는다** (디자인 노트 명시).

## 시나리오 4. 캘린더 → 상세 열람 (Flow D)

> 본인이 등록한 노하우를 월별 캘린더에서 둘러보고, 셀을 탭하여 상세를 본다.

### 전제 조건
- 시나리오 1 이 1회 이상 완료된 상태

### 플로우

**캘린더 진입**
1. 사용자: 하단 탭 "나의 기록" 으로 ARCHIVE 진입
2. 시스템: 현재 월 캘린더 + 통계(이번 달 기록 수 / 공유한 노하우 수) 노출
    - 셀: 사진 썸네일 배경, 우하단 흰점은 음성 기록 있음 표시

**월 이동**
3. 사용자: 상단 `< / >` 화살표로 이전·다음 월로 이동

**상세 열람**
4. 사용자: 캘린더 셀 탭 → DETAIL 화면 진입
5. 시스템: 사진 + 사용자 입력 제목 + 본문 영역(클라 하드코딩 `"안녕하세요"`) 노출. 위치 칩 / 음성 플레이어는 MVP 제외, 날짜는 상단 타이틀로 노출
6. 사용자: [수정] 또는 [공유하기] 분기
    - [수정]: 편집 화면으로 (MVP 범위 결정 필요)
    - [공유하기]: 공유 시트(SHARE_SHEET) 오픈 → 시나리오 2 의 분기 재사용
