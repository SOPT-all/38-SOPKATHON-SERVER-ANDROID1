# 공유 시트

| 항목 | 내용 |
|------|------|
| 화면 ID | SHARE_SHEET |
| 화면 위치 | RECORD 의 [공유] / DETAIL 의 [공유하기] 진입 |
| 작성일 | 2026/05/17 |

> 노하우 카드를 어디로 공유할지 선택하는 바텀시트. MVP 3개 옵션, 카카오톡 최우선.

---

## 컴포넌트

### 1. 배경
- 직전 화면(RECORD / DETAIL) dim 처리
- 외부 탭 시 시트 닫힘 (취소)

### 2. 시트
- 화면 하단에서 슬라이드업

#### 2-1. 시트 핸들
- 상단 중앙 짧은 가로 바

#### 2-2. 시트 타이틀
- "어디로 공유할까요?"

### 3. 공유 옵션 카드

#### 3-1. 카카오톡으로 보내기 (추천)
- 좌측 K 로고 + "카카오톡으로 보내기"
- 보조 텍스트 "자녀 · 가족방 · 친구에게"
- **추천 배지** 표시 (최우선 노출)
- 탭 → 카카오 공유 SDK 호출 (서버 호출 X)

#### 3-2. 링크 복사
- 좌측 🔗 아이콘 + "링크 복사"
- 보조 텍스트 "문자 · 다른 앱에 붙여넣기"
- 탭 → 카드 공유 URL 클립보드 복사 → 토스트 안내

#### 3-3. 또래 게시판에 공유
- 좌측 📣 아이콘 + "또래 게시판에 공유"
- 보조 텍스트 "같은 또래 분들과 나누기"
- 탭 → 노하우 카드를 isShared=true 로 게시 (서버 API 호출)

---

## 상호작용 / 비고

- 세 옵션 카드 색상은 동일 톤 (디자인 노트).
- 시트 외부 탭 시 닫힘.
- 서버 API 가 관여하는 옵션은 3-3 단 1건.
