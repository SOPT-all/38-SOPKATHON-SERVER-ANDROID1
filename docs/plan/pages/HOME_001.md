# 홈 (또래 공유 게시판)

| 항목 | 내용 |
|------|------|
| 화면 ID | HOME |
| 화면 위치 | 하단 탭 "홈" |
| 작성일 | 2026/05/17 |

> 또래 부모 사용자들이 isShared 로 게시한 노하우 카드를 갤러리 형태로 둘러본다.

---

## 컴포넌트

### 1. 상단 바
- 화면 최상단 고정

#### 1-1. 햄버거 메뉴
- 좌측 `☰` 아이콘

#### 1-2. 화면 타이틀
- 중앙 "노화우"

#### 1-3. 알림
- 우측 `🔔` 아이콘

### 2. 또래의 노하우 섹션 헤더
- "또래의 노하우" 라벨

### 3. 갤러리 카드 그리드
- 2×2 그리드 (전체 카드 한 번에 노출, MVP 페이지네이션 없음)

#### 3-1. 음성 길이 칩
- 카드 우상단, 마이크 아이콘 + 초 단위 (예: `30s`)

#### 3-2. 카드 비주얼
- 사진 또는 대표 이모지

#### 3-3. 작성자 정보
- 이름·나이 (예: "박순자님 · 68세")

#### 3-4. 제목 / 본문
- 상단: 사용자가 입력한 `title` (서버 응답)
- 하단 1~2줄 본문: 음성 녹음 STT 결과 자리. 클라가 받은 척 박은 `"안녕하세요"` 하드코딩 (서버 미수신·미보관)

#### 3-5. 좋아요 카운트
- `♥` 아이콘 + 숫자 — **클라 하드코딩** (서버 미보관)

#### 3-6. 댓글 카운트
- `💬` 아이콘 + 숫자 — **클라 하드코딩** (서버 미보관)

### 4. 하단 내비게이션
- 홈(활성) / 기록 / 나의 기록

---

## 상호작용 / 비고

- 카드 탭 → 상세(DETAIL) 이동은 **MVP 미구현** (디자인 노트).
- 페이지네이션 미적용 — 카드 전체를 한 번에 노출.
