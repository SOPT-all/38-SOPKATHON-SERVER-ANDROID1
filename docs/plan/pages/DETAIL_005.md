> 하나의 노하우 카드를 사진 + 위치/날짜 칩 + 제목 + 텍스트 변환문으로 펼쳐 본다.

---

## 컴포넌트

### 1. 상단 바
- 화면 최상단 고정

#### 1-1. 뒤로가기
- 좌측 `<` 아이콘

#### 1-2. 화면 타이틀
- 중앙 "{월} {일}일" (예: "5월 14일") — `createdAt` 기준

#### 1-3. 더보기
- 우측 `⋯` 아이콘

### 2. 사진 영역
- 큰 이미지
- 위치 칩(📍) 은 **MVP 제외** — 위치 정보 미저장
- 날짜 칩(📅) 은 화면 상단 타이틀("{월} {일}일") 로 갈음 — `createdAt` 기준

### 3. 제목
- 사용자가 입력한 `title` (서버 응답)
- 예: "상추 모종 심기"

### 4. 본문 (음성 기록 옮김)
- 음성 녹음 STT 결과 자리. 클라가 받은 척 박은 `"안녕하세요"` 하드코딩 (서버 미수신·미보관)

### 5. 액션 버튼
- 하단, 좌우 2분할

#### 5-1. [수정]
- 편집 화면 진입 (MVP 범위 결정 필요)

#### 5-2. [공유하기]
- 공유 시트(SHARE_SHEET) 오픈

### 6. 하단 내비게이션
- 홈 / 기록 / 나의 기록(활성)

---

## 상호작용 / 비고

- 디자인의 음성 플레이어(▶ 파형 0:30) 는 **MVP 제외** — 음성 파일 서버 미보관.
- 디자인의 위치 칩(📍 서울 노원구) 도 **MVP 제외** — 위치 정보 미저장.
- 날짜는 사진 EXIF 가 아닌 서버 `createdAt` 기준으로 화면 타이틀에 노출.
- 사진이 없는 카드의 경우 사진 영역은 플레이스홀더로 노출.
