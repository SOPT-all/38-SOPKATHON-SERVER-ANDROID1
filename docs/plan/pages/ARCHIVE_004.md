# 나의 노하우집 (월별 캘린더)

| 항목 | 내용 |
|------|------|
| 화면 ID | ARCHIVE |
| 화면 위치 | 하단 탭 "나의 기록" |
| 작성일 | 2026/05/17 |

> 사용자가 등록한 노하우 카드를 월별 캘린더로 한눈에 본다. 셀 = 사진 썸네일.

---

## 컴포넌트

### 1. 상단 바
- 화면 최상단 고정

#### 1-1. 화면 타이틀
- 중앙 "나의 노하우집"

#### 1-2. 설정
- 우측 `⚙` 아이콘

### 2. 월 헤더

#### 2-1. 월 라벨
- "2026년 5월" 등 현재 표시 월

#### 2-2. 월 이동 화살표
- 우측 `< >` 두 버튼
- 탭 시 이전·다음 월로 이동

### 3. 통계 카드
- 가로 정렬 카드 묶음

#### 3-1. 이번 달 기록
- 큰 숫자 + 라벨 "이번 달 기록"

#### 3-2. 공유한 노하우
- 큰 숫자 + 라벨 "공유한 노하우" (isShared=true 카운트)

### 4. 캘린더 그리드

#### 4-1. 요일 헤더
- 일 / 월 / 화 / 수 / 목 / 금 / 토

#### 4-2. 날짜 셀
- 좌상단에 날짜 숫자
- 사진 썸네일이 셀 배경에 깔림
- 셀 색상 = 사진 대표색 (클라가 썸네일에서 자체 산출)

#### 4-3. 셀 인터랙션
- 탭 시 해당 날짜의 상세 페이지(DETAIL) 진입

### 5. 하단 내비게이션
- 홈 / 기록 / 나의 기록(활성)

---

## 상호작용 / 비고

- 디자인의 검은 박스 영역은 미공개 / 후순위로 간주.
- 한 날짜에 다수 기록이 있을 경우의 셀 처리는 디자인 미확정 — 임시로 최신 1건 썸네일.
