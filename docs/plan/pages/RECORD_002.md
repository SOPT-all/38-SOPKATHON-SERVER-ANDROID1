# 기록 (오늘의 기록)

| 항목 | 내용 |
|------|------|
| 화면 ID | RECORD |
| 화면 위치 | 하단 탭 "기록" |
| 작성일 | 2026/05/17 |

> 한 화면에서 제목 · 사진 · 음성 기록 · 저장 분기를 모두 처리한다. 3탭 이내 저장 완료.

---

## 컴포넌트

### 1. 상단 바
- 화면 최상단 고정

#### 1-1. 닫기 버튼
- 좌측 `×` 아이콘
- 탭 시 이전 화면으로 복귀 (입력 중이면 확인)

#### 1-2. 화면 타이틀
- 중앙 "오늘의 기록"

#### 1-3. 더보기
- 우측 `⋯` 아이콘

### 2. 제목 입력
- 텍스트 필드 ("기록 제목")

### 3. 사진 첨부 영역
- 큰 박스, 중앙에 📷 아이콘 + "사진을 찍어주세요" placeholder

#### 3-1. 사진 선택/촬영
- 탭 시 시트 노출 → 촬영 / 갤러리 선택

### 4. 음성 기록 영역
- 빨간 원형 녹음 버튼

#### 4-1. 녹음 안내
- 라벨 "음성 기록 / 길게 눌러 녹음하세요"

#### 4-2. 파형 시각화
- 녹음 중 음성 파형 표시

#### 4-3. 처리 방식
- 클라가 녹음 UX 만 수행하고, STT 결과는 받은 척 `"안녕하세요"` 를 하드코딩으로 보관.
- 서버 전송 / DB 보관 모두 **없음** — 클라 변수로만 유지하며 HOME 카드 본문·DETAIL 본문에 그대로 노출.
- 길이(초) 만 서버로 전송하여 카드 우상단 음성 길이 칩 노출에 사용.

### 5. 액션 버튼
- 하단, 좌우 2분할

#### 5-1. [저장만]
- 비공개 저장 (isShared=false)
- 저장 완료 → 토스트 + 이전 화면 복귀

#### 5-2. [공유]
- 공유 시트(SHARE_SHEET) 오픈
- 시트에서 "또래 게시판에 공유" 선택 시 isShared=true 로 등록

### 6. 하단 내비게이션
- 홈 / 기록(활성) / 나의 기록

---

## 상호작용 / 비고

- 등록 실패 시 토스트로 재시도 안내, 입력 유지.
- 사진 EXIF 의 위치/촬영 시각은 MVP 에서 사용하지 않음 — 저장·전송 모두 X.
