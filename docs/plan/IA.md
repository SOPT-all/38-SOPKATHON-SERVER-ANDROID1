# 노화우 — Information Architecture

| 항목 | 내용 |
|------|------|
| 서비스 | 노화우 (老하우) |
| 한 줄 소개 | 노화가 다가오는 5·60대를 위해 건강한 나이듦의 노하우를 제안하는 서비스 |
| 1차 사용자 | 5·60대 부모 세대 (기록자 / 또래 공유 참여자) |
| 2차 사용자 | 2030~40대 자녀 (수신자 — 카카오톡 / 링크로 수신) |
| 컨셉 | Non-AI = 자연 / "노하우집" |
| 작성일 | 2026-05-17 |

> 화면 5종 (홈 / 기록 / 공유 시트 / 나의 노하우집 / 상세) + 하단 3탭 내비게이션 (홈·기록·나의 기록).

---

## 화면 목록

| 화면 ID | 화면명 | 진입 경로 |
|---------|--------|-----------|
| HOME | 홈 (또래 공유 게시판) | 하단 탭 "홈" |
| RECORD | 기록 (오늘의 기록) | 하단 탭 "기록" |
| SHARE_SHEET | 공유 시트 (바텀시트) | RECORD 의 [공유] 버튼 / DETAIL 의 [공유하기] |
| ARCHIVE | 나의 노하우집 (월별 캘린더) | 하단 탭 "나의 기록" |
| DETAIL | 상세 페이지 | ARCHIVE 캘린더 셀 탭 (홈 카드 탭은 MVP 미구현) |

## HOME — 홈 / 또래 공유 게시판

- 상단 바
    - ㄴ 햄버거 (`☰`)
    - ㄴ 화면 타이틀 "노화우"
    - ㄴ 알림 (`🔔`)
- "또래의 노하우" 섹션 헤더
- 갤러리 카드 (2×2 그리드)
    - ㄴ 우상단 음성 길이 칩 (예: `30s`)
    - ㄴ 사진 / 대표 이모지
    - ㄴ 작성자 이름·나이 (예: "박순자님 · 68세")
    - ㄴ 제목 (사용자 입력 `title`)
    - ㄴ 본문 1~2줄 — 음성 STT 결과 자리, 클라 하드코딩 `"안녕하세요"` (서버 미수신)
    - ㄴ 좋아요 카운트 (`♥ 24`) — **클라 하드코딩**
    - ㄴ 댓글 카운트 (`💬 8`) — **클라 하드코딩**
- 하단 내비게이션 (홈 활성)

> 인터랙션: 카드 탭 → 상세(DETAIL) **이동은 MVP 미구현** (디자인 노트: "홈 게시판 카드 클릭 시 상세 페이지는 목업·구현 X"). MVP 에서는 페이지네이션 없이 카드 전체를 한 번에 반환한다.

---

## RECORD — 기록 / 오늘의 기록

- 상단 바
    - ㄴ 닫기 (`×`)
    - ㄴ 화면 타이틀 "오늘의 기록"
    - ㄴ 더보기 (`⋯`)
- 제목 입력 텍스트 필드
- 사진 첨부 영역
    - ㄴ 카메라 아이콘 + "사진을 찍어주세요" placeholder
    - ㄴ 탭 시 촬영 / 갤러리 선택
- 음성 기록 영역
    - ㄴ 빨간 원 녹음 버튼 ("길게 눌러 녹음하세요")
    - ㄴ 음성 파형 시각화
    - ㄴ 녹음 종료 후 STT 로 텍스트 변환
- 액션 버튼
    - ㄴ [저장만] — 비공개 저장
    - ㄴ [공유] — 공유 시트(SHARE_SHEET) 오픈
- 하단 내비게이션 (기록 활성)

> 핵심: 한 화면에 사진·음성·태깅이 동시에. 3탭 이내 저장 완료.

---

## SHARE_SHEET — 공유 시트 (바텀시트)

- 배경: 직전 화면(RECORD / DETAIL) dim 처리
- 시트 핸들 + 타이틀 "어디로 공유할까요?"
- 옵션 카드 (3개)
    - ㄴ 카카오톡으로 보내기 (자녀·가족방·친구) — **추천 배지**
    - ㄴ 링크 복사 (문자·다른 앱에 붙여넣기)
    - ㄴ 또래 게시판에 공유 (같은 또래 분들과 나누기)

> MVP 옵션 3개. 카카오톡 최우선 (추천 배지). 시트 외부 탭 시 닫힘.
> 서버 API 가 필요한 항목은 "또래 게시판에 공유" 1건 — RECORD 등록 시 `isShared` 본문 필드로 처리.

---

## ARCHIVE — 나의 노하우집 (월별 캘린더)

- 상단 바
    - ㄴ 화면 타이틀 "나의 노하우집"
    - ㄴ 설정 (`⚙`)
- 월 헤더
    - ㄴ "2026년 5월"
    - ㄴ 이전 / 다음 월 화살표 (`< >`)
- 통계 카드
    - ㄴ "이번 달 기록" 수
    - ㄴ "공유한 노하우" 수
- 캘린더 그리드 (요일 헤더 + 셀)
    - ㄴ 사진 썸네일이 셀에 깔림
    - ㄴ 셀 색상 = 사진의 대표색 (클라가 썸네일에서 자체 산출)
- 하단 내비게이션 (나의 기록 활성)

> 인터랙션: 셀 탭 → 상세(DETAIL).

---

## DETAIL — 상세 페이지

- 상단 바
    - ㄴ 뒤로가기 (`<`)
    - ㄴ 화면 타이틀 "{월} {일}일"
    - ㄴ 더보기 (`⋯`)
- 사진 영역 (큰 이미지)
    - ㄴ 위치 칩 / 날짜 칩은 **MVP 제외** (위치 미저장, 날짜는 상단 타이틀 `createdAt` 으로 갈음)
- 제목 (사용자 입력 `title`)
- 본문 ("음성 기록 옮김") — 음성 STT 결과 자리, 클라 하드코딩 `"안녕하세요"` (서버 미수신)
- 액션 버튼
    - ㄴ [수정]
    - ㄴ [공유하기] → 공유 시트(SHARE_SHEET)
- 하단 내비게이션 (나의 기록 활성)

> 디자인의 음성 플레이어(▶ 파형 0:30) 는 **MVP 제외** — 음성 파일을 서버 보관하지 않음. 텍스트 변환문만 표시.
> 위치 칩도 **MVP 제외** — 위치 정보 미저장. 날짜는 상단 타이틀의 `createdAt` 으로 갈음.

---

## 하단 내비게이션 (공통 3탭)

- 홈 (HOME)
- 기록 (RECORD)
- 나의 기록 (ARCHIVE)

---

## 화면 이동 다이어그램

```
[홈 HOME] ─ 카드 전체 노출 / 카드 탭(MVP 미구현)
   │
   ├── (탭) ─→ [기록 RECORD] ─ [저장만] ─→ 카드 저장 (isShared=false)
   │                            └ [공유] ─→ [공유 시트 SHARE_SHEET]
   │                                          ├── 카카오톡 (외부)
   │                                          ├── 링크 복사 (외부)
   │                                          └── 또래 게시판 (isShared=true)
   │
   └── (탭) ─→ [나의 기록 ARCHIVE] ─ 셀 탭 ─→ [상세 DETAIL]
                                                 ├ [수정]
                                                 └ [공유하기] ─→ [공유 시트]
```
